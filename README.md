# Mythlight
Mythlight is a fledging project intended to become an open-source game as it begins development. 
As of now, it is not ready for additional contributors or any sort of actual play. Whatever you see in progress currently is the core of the game in its earliest steps.

When Mythlight opens for contributors, please ensure you are branching off of the **develop** branch. Any pull requests going directly to master or branches directly off of master will be rejected.
